# CodeAlpha FAQ Chatbot

## Project Description

This project is developed as part of the CodeAlpha AI Internship.

The FAQ Chatbot answers user questions based on predefined frequently asked questions using AI and NLP techniques.

## Features

* User can ask questions
* Matches questions using NLP
* Returns best matching answer
* Interactive chatbot UI

## Technologies Used

* Python
* Scikit-learn
* Google Colab
* ipywidgets
